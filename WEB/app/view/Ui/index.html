概述

设计目的：标准化、体验统一、简单可靠、易于扩展

后端PHP框架

基于PHP、smarty 构建。
基于composer自动加载。
完全基于API接口设计，API文档自动生成。
权限包含菜单访问权限和API接口访问权限。
工具类、数据库操作封装。
默认用户：super_admin 密码：域名

前端H5框架

兼容webkit内核最新浏览器，不兼容老式浏览器。
基于jquery、Bootstrap4、layer 弹出层
设计基于原始HTML5组件，自定义的表格和表单控件。
前后端交互标准封装，细节优化实现快速开发。
降低前端开发难度

工具依赖

图标选择 fontawesome Bootstrap4 layer 弹出层 popper 弹出层 jquery 360CDN smarty 模板引擎

第三方组件

EChart chart.js jstree leafletjs 地图
